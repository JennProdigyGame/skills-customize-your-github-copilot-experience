# 📘 Assignment: Hello World in Python

## 🎯 Objective

Write a simple Python program that prints a greeting message. This assignment introduces basic Python syntax, script structure, and running code from the command line.

## 📝 Tasks

### 🛠️ Create a Hello World script

#### Description
Create a Python file named `hello_world.py` that prints a friendly greeting message when executed. Ensure that the program is easy to read, properly formatted, and runs without errors.

#### Requirements
Completed program should:

- Define a function named `main()` that prints `Hello, world!` or similar greeting
- Call `main()` when module is run as a script (using `if __name__ == '__main__':`)
- Run in terminal with `python hello_world.py` and output the greeting message
- Include comments that explain what the code does
